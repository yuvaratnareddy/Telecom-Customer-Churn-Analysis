
# 📊 Telecom Customer Churn Analysis

---

## 🔹 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on telecom customer data to identify patterns influencing customer churn.  
Customer churn analysis helps businesses understand why customers leave and enables data-driven retention strategies.

---

## 🎯 Problem Statement
Telecom companies face customer loss due to service dissatisfaction, pricing, and contract types.  
The goal of this project is to analyze customer data and discover factors contributing to churn.

---

## 📂 Dataset Description

The dataset contains telecom customer information including:

| Feature Category | Description |
|------------------|-------------|
| Demographics | Gender, Senior Citizen status |
| Services | Phone, Internet, Streaming services |
| Account Info | Contract type, tenure |
| Billing | Monthly charges, total charges |
| Target Variable | Churn (Yes / No) |

Dataset File:
```

TelecomCustomerChurn.csv

````

## 📑 Project Documentation

### 1️⃣ Data Collection
- Imported telecom dataset into Jupyter Notebook.
- Verified dataset size and column structure.

### 2️⃣ Data Understanding
- Used `.info()` and `.describe()` for statistical summary.
- Identified categorical and numerical variables.

### 3️⃣ Data Cleaning
- Handled missing values.
- Converted data types where required.
- Removed inconsistencies in categorical fields.

### 4️⃣ Exploratory Data Analysis (EDA)
Performed analysis to understand:
- Customer churn distribution
- Contract type impact
- Monthly charges vs churn
- Tenure relationship with retention

### 5️⃣ Visualization Techniques
Created visualizations such as:
- Bar Charts
- Histograms
- Box Plots
- Scatter Plots

These visuals helped identify patterns and anomalies.

---

## 📊 Key Findings
- Month-to-month contracts show higher churn rates.
- Customers with higher monthly charges churn more frequently.
- Longer tenure customers are more loyal.
- Service bundles influence retention behavior.
